# CodeFix AI 🛠️ - Estación de Auditoría de Código con IA

**CodeFix AI** es una solución de software diseñada para optimizar los procesos de soporte técnico y revisión de código remotos. El sistema integra modelos de lenguaje avanzados para escanear, estructurar y diagnosticar errores de sintaxis en tiempo real, devolviendo al usuario la línea exacta de la falla y el bloque de código completamente corregido.

---

## 🚀 Demostración y Acceso Directo

* **Aplicación en Vivo (Frontend):** [Clusa aquí para probar CodeFix AI desde tu móvil](TU_LINK_DE_GITHUB_PAGES_AQUÍ)
* **Servidor de Procesamiento (Backend):** Desplegado en la nube mediante Render.

---

## 🧠 Características Principales

* **Escaneo Dinámico por Líneas:** El frontend segmenta y numera automáticamente el código del usuario antes del envío para un análisis milimétrico.
* **Diagnóstico de Precisión con IA:** Integración directa con el modelo **Gemini 2.5 Flash** a través de su API oficial para auditorías automáticas de software.
* **Dictado Nativo por Voz:** Incorporación de la interfaz *Web Speech API* para permitir el ingreso de bloques de código y comandos técnicos mediante el micrófono del dispositivo móvil.
* **Interfaz PWA Optimizada:** Diseño responsive en Modo Oscuro (Dark Mode), configurable para ejecutarse en pantalla completa (*Standalone*) en dispositivos Android e iOS como una app nativa.

---

## 🛠️ Tecnologías Utilizadas

* **Backend:** Node.js, Express.js (Arquitectura de API REST)
* **Inteligencia Artificial:** SDK oficial de `@google/genai` (Google GenAI)
* **Frontend:** HTML5, CSS3 (Diseño fluido y accesible), JavaScript Moderno (Async/Await, Fetch API)
* **Seguridad:** Gestión de variables de entorno mediante `dotenv` y políticas CORS integradas.

---

## 📈 Impacto y Casos de Uso
Esta herramienta fue desarrollada con el objetivo de servir como un recurso de primera línea para:
1.  **Células de soporte técnico:** Reducción del tiempo de diagnóstico de errores comunes (Bugs) de horas a segundos.
2.  **Desarrolladores Junior/Freelance:** Herramienta de auto-auditoría remota antes de subir cambios a producción.
